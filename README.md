# 运行

## 以SpringBoot方式运行

本项目为Spring Boot`项目，故支持常规Spring Boot项目运行方式

## 以Docker方式运行

可以把Spring Boot和Docker结合起来，把运行环境封装在镜像里面，具体构建方式请参考API文档
